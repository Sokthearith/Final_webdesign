# Bookshop - Online Bookstore

A modern, responsive online bookstore web application featuring a curated collection of classic and contemporary books.

## Features

- **Book Catalog**: Browse 12 books with category filtering (All, Classic, Fiction, Fantasy)
- **Book Details**: View detailed information, ratings, and descriptions
- **Shopping Cart**: Add books, manage quantities, view subtotal
- **User Authentication**: Login/Sign up system with session management
- **Responsive Design**: Works on desktop, tablet, and mobile devices

## Technologies

- HTML5, CSS3, JavaScript (Vanilla)
- LocalStorage for cart and session management

## Getting Started

1. Place book cover images in `bookshop/assets/images/` 
2. Open `bookshop/index.html` with Live Server or any web server
3. Browse books, login, and add items to cart

## Project Structure

```
bookshop/
├── assets/
│   ├── css/styles.css
│   ├── js/ (data.js, script.js, book-detail.js, login.js, shared.js)
│   └── images/ (12 book cover images)
├── index.html
├── book-detail.html
└── login.html
```

## Usage

- Browse books on the homepage
- Click any book to view details
- Login to add books to cart
- Manage cart items and quantities
